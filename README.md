# TCC: Otimização de Permutação via Heurísticas do Cubo Mágico

Este repositório contém a implementação e documentação do Trabalho de Conclusão de Curso (TCC) focado na aplicação de algoritmos de resolução do Cubo Mágico (como o Algoritmo de Duas Fases de Kociemba) para resolver problemas de otimização combinatória.

## Estrutura do Projeto

*   `docs/`: Documentação detalhada, cronograma e relatórios.
*   `src/`: Código-fonte das implementações.
    *   `models/`: Modelagem de problemas (TSP, Job Scheduling, etc.).
    *   `algorithms/`: Implementação das heurísticas inspiradas no Cubo Mágico.
*   `data/`: Instâncias de teste e resultados de experimentos.
*   `tests/`: Testes unitários e de integração.

## Metodologia e Cronograma

O projeto segue um cronograma de 6 meses, detalhado em [docs/cronograma.md](docs/cronograma.md).
