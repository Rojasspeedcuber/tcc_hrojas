# Cronograma Sugerido (6 Meses)

| Mês | Fase | Atividades Principais |
| :--- | :--- | :--- |
| 1 | Pesquisa e Modelagem | Revisão bibliográfica (Kociemba, Teoria dos Grupos). Definição dos subgrupos análogos para o problema escolhido. |
| 2 | Modelagem Técnica | Formalização matemática do estado e movimentos. Mapeamento de instâncias de teste. |
| 3 | Desenvolvimento Base | Implementação da estrutura de dados e operadores de movimento básicos em Python. |
| 4 | Implementação da Heurística | Codificação do algoritmo de duas fases. Criação de tabelas de lookup/pruning. |
| 5 | Experimentação | Execução de testes com instâncias reais (TSP/Scheduling). Análise comparativa de desempenho. |
| 6 | Redação e Defesa | Finalização do texto do TCC. Preparação dos slides e defesa. |
